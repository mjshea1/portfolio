Game of Chess

Game of Chess is a sleek, modern, and fun iPhone App created to utilize many features within iOS 8. It is an experimental, flat, game design for a turn based chess game played with friends. The app - including the logo, graphics, and game design - was created in a clean and modern design to enhance and simplify the user experience. In life, as in chess, forethought wins. - Charles Buxton

iPhone App - September 2014
